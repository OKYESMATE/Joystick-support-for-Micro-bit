# Joy-bit
An extension for Micro-bit V1 and V2 that allows joy stick connection via pins

# How to install?
Microsoft Makecode:
1. Create or load the project of choice
2. In the find the settings icon (it should look like a gear and it should be in the top-right corner) and click it
3. Find the extensions button and click it
4. In the search bar paste this exact link: https://github.com/OKYESMATE/joy-bit
5. Click enter
6. You should see an extension named "joystick-input" or joy-bit, click it to add the extension
7. Viola! you have loaded and installed the extension and is now ready for use.
8. Reference the "How to use" section in this readme to find out how to use the extension!
